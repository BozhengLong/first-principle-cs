# Guide 指导文档

本目录包含项目的核心指导文档，解释我们为什么做这个项目、如何规划、以及包含哪些系统。

## 文档索引

- [WHY.md](./WHY.md) - 项目动机与"理解"的定义
- [ROADMAP.md](./ROADMAP.md) - 12 个月开发路线图
- [PROJECTS.md](./PROJECTS.md) - 核心系统仓库清单

## 使用说明

如果你是第一次接触这个项目，建议按以下顺序阅读：

1. 先读 [WHY.md](./WHY.md) 理解项目的哲学基础
2. 再看 [PROJECTS.md](./PROJECTS.md) 了解我们要构建什么
3. 最后查看 [ROADMAP.md](./ROADMAP.md) 了解实施计划

如果你想参与贡献，请同时阅读 [../principles/](../principles/) 目录下的原则文档。
